# 💰 Control de Gastos - Budget Tracker

Una aplicación intuitiva de gestión financiera personal que permite definir un presupuesto, registrar gastos por categorías y visualizar el consumo mediante indicadores gráficos.


## 🎯 Propósito del Proyecto

El objetivo es ayudar a los usuarios a tener un control estricto de sus finanzas. La app permite visualizar cuánto dinero se ha gastado, cuánto queda disponible y qué porcentaje del presupuesto total ha sido utilizado mediante una interfaz dinámica.

## ✨ Funcionalidades Principales

- **Gestión de Presupuesto:** Define un presupuesto inicial para comenzar el seguimiento.
- **Control de Gastos:** Añade, edita y elimina gastos con facilidad.
- **Categorización Intuitiva:** Clasifica tus gastos (Ahorro, Comida, Casa, Gastos Varios, Ocio, Salud, Suscripciones).
- **Filtros Inteligentes:** Filtra tus registros por categoría para analizar en qué gastas más.
- **Visualización Gráfica:** Gráfico circular interactivo que muestra el porcentaje del presupuesto consumido.
- **Alertas de Presupuesto:** Indicadores visuales cuando el gasto se aproxima o supera el límite disponible.
- **Persistencia de Datos:** Los datos se mantienen seguros en el `localStorage` del navegador.

## 🛠️ Tecnologías Utilizadas

- **React.js**: Para la construcción de la interfaz.
- **TypeScript**: Para un desarrollo seguro y tipado de los datos financieros.
- **Context API**: Para la gestión del estado global de la aplicación.
- **Tailwind CSS**: Para un diseño limpio, moderno y adaptable.
- **React Circular Progressbar**: Para la visualización de datos estadísticos.
- **Datepicker & Modals**: Para una experiencia de usuario fluida al registrar datos.
